# 🎓 Student Performance Prediction System

## Overview
AI-powered system that predicts student performance and risk levels with 97.81% accuracy.

## Model Performance
- Performance Accuracy: 97.81%
- Risk Level Accuracy: 99.29%
- F1-Score: 0.98

## Features
- Predict Performance (Poor/Average/Good)
- Predict Risk Level (High/Medium/Low)
- Single & Batch Prediction
